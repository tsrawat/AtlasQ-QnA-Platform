# AtlasQ

AtlasQ is a clean full-stack college project for asking questions, posting answers, voting, searching, and viewing user profiles.

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Appwrite authentication, database, and storage

## Features

- Email/password authentication
- Ask and edit questions
- Upload question images
- Answer questions
- Upvote and downvote
- Search and tag filtering
- User profile pages

## Environment Variables

Create `.env.local` from `.env.sample`:

```bash
NEXT_PUBLIC_APPWRITE_HOST_URL=https://cloud.appwrite.io/v1
NEXT_PUBLIC_APPWRITE_PROJECT_ID=your_project_id
APPWRITE_API_KEY=your_server_api_key
```

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Checks

```bash
npm run lint
npx tsc --noEmit
npm run build
```
